This in-browser version of the popular Magic the Gathering card game took me a little under a week to build from scratch using Node.js, Socket.io, Express, and jQuery. It requires a mouse, keyboard, and a minimum resolution of 900x500 to play. This project is going to be integrated to my social media website to track user statistics, and to a database to view cards and create decks. Start a game by opening two demo tabs and choosing the decks.
